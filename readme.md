# Dayal Optical Website 👓

## 📌 Project Description

This project is a responsive website developed using **HTML, CSS, and Bootstrap**.
It is designed for an optical shop to showcase products, services, and contact details in a clean and user-friendly interface.

---

## 🚀 Technologies Used

* HTML5
* CSS3
* Bootstrap 5

---

## ✨ Features

* Responsive design (Mobile + Desktop friendly)
* Modern UI using Bootstrap
* Navigation bar
* Image carousel / banner
* Product section
* Contact form
* Footer section

---

## ▶️ How to Run the Project

1. Download or clone this repository
2. Open the folder
3. Double click **index.html**
4. Website will open in your browser
